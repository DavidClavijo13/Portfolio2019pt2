Discord Clone

What It Does:

This is a messagin app that works with Socket.io to deliver real time information instead of using CRUD operations to communicate with a server.

What I Learned:

I did not utilize a state library, so I learned how to prop-drill.. a lot! Socket.io was quite the learning experience as well. The server would crash whenever a user unexpectedly left, and I thought it would be the end of me. Luckily I was able to fix this by implementing a reconnect function to check for current users and reconnect them.

Tech Used:

React, Socket.io, Heroku, NPM
